## [1.0.0] - 2026-03-24

### Added
- Initial release of xul/laravel-project-map
- Artisan command: `project:map`
- Recursive directory tree builder
- Configurable depth, exclusions, and file inclusion
- Support for JSON and text output formats
- Ability to save output to file
- Vendor and node_modules toggles
- Hidden file support
- Sorting configuration
- Full Pest test suite
